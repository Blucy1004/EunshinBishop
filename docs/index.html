Core systems

Modern search. Clear structure.

01

Principal Variation Search

Alpha-beta search with PVS, iterative deepening, aspiration windows, and transposition-table support.

02

Selective Search

Null-move pruning, late-move reductions, move ordering, internal iterative reduction, and quiescence search.

03

NNUE Evaluation

Supports neural evaluation through a configurable NNUE network, including residual and absolute output modes.

04

AEGIS

Adaptive Evaluation Guard for Instability in Search reduces aggressive search reductions when a position appears unstable.

05

LIMBO

Localized Instability Monitor and Bounded Deepening Override adds limited verification around unstable frontier nodes.

Using the engine

Bring your own chessboard.

EunshinBishop Q is a standard UCI engine. It can be loaded into a compatible chess GUI, or played directly through the browser playroom.

i

The browser playroom currently uses a JavaScript engine adapter. Its interface is prepared for the official EunshinBishop WebAssembly engine.

Basic setup UCI
  1. 01 Download and extract the latest release.
  2. 02 Open a UCI-compatible chess GUI.
  3. 03 Add EunshinBishop.exe as a new engine.
  4. 04 Configure the NNUE file and optional search settings.
